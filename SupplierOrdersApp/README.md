# SupplierOrdersApp

WinForms-приложение на C# для учета заказов магазина поставщикам.

## Возможности

- Создание заказов поставщикам с товарами, количеством, ценой и общей суммой.
- Отметка подтверждения заказа поставщиком.
- Внесение оплат по заказу.
- Контроль правила оплаты: подтвержденный заказ должен быть полностью оплачен в течение 30 дней.
- Материальный отчет за период: поставщик, товар, количество, общая сумма, оплаченная сумма.
- Экспорт материального отчета в `.xlsx`.
- Таблица и круговая диаграмма за период по выбранным поставщикам:
  - сумма заказов;
  - сумма отказов/просрочек, то есть неоплаченная часть подтвержденных заказов после 30 дней.

## PostgreSQL

Создайте базу данных, например:

```sql
create database supplier_orders;
```

Строка подключения зашита в приложении:

```text
Host=localhost;Port=5432;Database=supplier_orders;Username=postgres;Password=1234
```

Если пароль или имя пользователя отличаются, измените константу `ConnectionString` в `Form1.cs`.

## Запуск

```powershell
dotnet restore
dotnet run
```

В приложении:

1. На главном экране откройте `База данных`.
2. Нажмите `Проверить подключение`.
3. Нажмите `Создать таблицы`.
4. При необходимости нажмите `Заполнить примером`.
5. Вернитесь назад и откройте нужный раздел:
   - `Управление таблицами`;
   - `Заказы и оплаты`;
   - `Материальный отчет`;
   - `Диаграмма заказов и отказов`.

Раздел `Управление таблицами` закрывает требования зачетной задачи:

- показывает данные таблиц `suppliers`, `products`, `purchase_orders`, `purchase_order_items`, `payments`;
- добавляет, изменяет и удаляет записи;
- изменение и удаление выполняются по выделенной строке таблицы;
- связанные поля выбираются из списков по названиям, а не по ID;
- заказы и состав выбранного заказа находятся на одной форме и связаны между собой.

Формы разделены по файлам:

- `Form1.cs` и `Form1.Designer.cs` - главное меню.
- `DatabaseSetupForm.cs` и `DatabaseSetupForm.Designer.cs` - подготовка базы данных.
- `OrdersForm.cs` и `OrdersForm.Designer.cs` - заказы, подтверждение и оплаты.
- `MaterialReportForm.cs` и `MaterialReportForm.Designer.cs` - материальный отчет и экспорт в Excel.
- `RatioReportForm.cs` и `RatioReportForm.Designer.cs` - таблица и круговая диаграмма заказов/отказов.
